CAILogin

Counter Artificial Intelligence Login


CAILogin is an image-based user verification program for web-page logins. It's written in plain C with no special dependencies, and hence it's very suitable for embedded systems which lack "big" interpreted languages, such as PHP, Python or Perl. It features a command line interface, which allows it to be very seamlessly integrated to shell scripts.


It works by generating true random numbers, on which applies several obscuration filters. The end result is an image from which is very difficult to extract the displayed number, by utilizing current AI technologies.


See a screenshot. For reasons unrelated to CAILogin (probably apache's restrictions), the example login form, doesn't work. If any of the filters/transformations seem to you too fancy, you can remove them at compile time.
